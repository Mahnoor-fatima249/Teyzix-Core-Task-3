## 🎯 Executive Summary
The **Teyzix-Core-Task-3** is a sophisticated, AI-augmented invoice management solution. It bridges the gap between traditional manual billing and modern digital verification by integrating **Generative AI** with dynamic **QR-based security**.

## 🚀 Deployment & Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Mahnoor-fatima249/Teyzix-Core-Task-3.git](https://github.com/Mahnoor-fatima249/Teyzix-Core-Task-3.git)

```

2. **Install dependencies:**
```bash
npm install --legacy-peer-deps

```


3. **Configure Environment:** Create `.env.local` and add your `GROQ_API_KEY`.
4. **Launch:**
```bash
npm run dev

```
